## 一、基础理论

### 第01节 TTS介绍

**TTS是Text To Speech的缩写，即“从文本到语音”，是人机对话的一部分，让机器能够说话。**

```
TTS是语音合成应用的一种，它将储存于电脑中的文件，如帮助文件或者网页，转换成自然语音输出。
TTS不仅能帮助有视觉障碍的人阅读计算机上的信息，更能增加文本文档的可读性。
TTS应用包括语音驱动的邮件以及声音敏感系统，并常与声音识别程序一起使用。
```

## 二、案例代码

### 第01节  工具类

```java
import android.content.Context;
import android.speech.tts.TextToSpeech;

import java.util.Locale;

public class CommonTTSUtil {

    private static TextToSpeech tts;
    private static final String TAG = CommonTTSUtil.class.getName();

    /***
     * TTS播报工具方法
     *
     * @param context   上下文对象
     * @param locale    本地语言设置 eg: Locale.CHINA
     * @param message   需要播报的文字信息
     */
    public static void openTtsPlay(Context context, Locale locale, String message) {
        tts =
                new TextToSpeech(
                        context,
                        status -> {
                            if (status == TextToSpeech.SUCCESS) {
                                tts.setLanguage(locale);
                                tts.speak(message, TextToSpeech.QUEUE_FLUSH, null, null);
                            }
                        });
    }
}
```





### 第02节 测试类

播放中文信息

```java
CommonTTSUtil.openTtsPlay(context, Locale.CHINA, "您已经进入夜间模式, 请及时打开车灯");
```

播放英文信息

```java
CommonTTSUtil.openTtsPlay(context, Locale.ENGLISH, "You have entered night mod");
```







### 第03节 清单文件

当前信息，需要存放到 清单文件中，与 用户权限设置在同一级别中。 否则会导致 TTS 播报失效

```xml
  <!--  TTS 在 Android 11 版本当中, 需要使用文字转语音的能力  -->
  <queries>
      <intent>
          <action android:name="android.intent.action.TTS_SERVICE" />
      </intent>
  </queries>
```
