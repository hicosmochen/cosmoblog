## 第一章 形状 Shape

### 第01节 基础使用

说明

```
在安卓中，shape（形状）是一种可用于定义视图背景的 XML 文件。
它可以用于为按钮、文本框、布局等视图设置各种形状和样式。
使用 shape 可以创建圆角矩形、椭圆、矩形、线条等各种形状，并可以自定义填充颜色、边框颜色、边框宽度等属性
```

API

```
1、 shape	指明当前的形状
2、 solid 	指明填充的颜色
3、 corners  指明圆角的半径
4、 stroke  	边框的颜色和宽度
5、 gradient 指明渐变色的颜色 （备注：如果指明渐变色之后，填充颜色可能会失效）
6、 size    	指明形状的宽度和高度 （备注：如果shape作为其他View的背景, 其他View设置了固定宽高, 那么size会失效）
```





### 第02节 案例代码

【一】圆角矩形图形

```xml
<?xml version="1.0" encoding="utf-8"?>
<shape xmlns:android="http://schemas.android.com/apk/res/android">

    <!--
    设置填充的颜色
    -->
    <solid android:color="@color/green" />

    <!--
     设置圆角的半径
    -->
    <corners android:radius="@dimen/dp_0010" />

    <!--
     设置边框的宽度和颜色
    -->
    <stroke
        android:width="@dimen/dp_0003"
        android:color="@color/argb_FFFF9900" />

    <!--
    设置形状的宽度和高度
    说明: 如果当前的 shape 作为背景给其他View, 如果其他View设置了固定的宽度和高度, 那么这里的size将会失效
    -->
    <size
        android:width="@dimen/dp_0060"
        android:height="@dimen/dp_0060" />

</shape>
```

【二】圆形的渐变图

```xml
<?xml version="1.0" encoding="utf-8"?>
<shape xmlns:android="http://schemas.android.com/apk/res/android" android:shape="oval">

    <!--
     设置圆角的半径
    -->
    <corners android:radius="@dimen/dp_0010" />

    <!--
     设置边框的宽度和颜色
    -->
    <stroke
        android:width="@dimen/dp_0003"
        android:color="@color/argb_FFFF9900" />

    <!--
    设置形状的宽度和高度
    说明: 如果当前的 shape 作为背景给其他View, 如果其他View设置了固定的宽度和高度, 那么这里的size将会失效
    -->
    <size
        android:width="@dimen/dp_0060"
        android:height="@dimen/dp_0060" />


    <!--  设置渐变色  -->
    <gradient
        android:centerColor="@color/argb_FF2A7E66"
        android:endColor="@color/argb_FF00BFFF"
        android:startColor="@color/argb_FFF0E68C"
        android:type="linear" />


</shape>
```

## 第二章 选择器 selector

### 第01节 基础使用

说明

```
在安卓中，selector（选择器）是一种可用于定义视图状态的 XML 文件。
它可以根据不同的视图状态（如按下、选中、禁用等）应用不同的样式或背景。
使用 selector 可以在不同的状态下为视图提供不同的外观，使用户界面更具交互性。
```

API

```
一、基础状态
1、normal（默认状态）：控件未被触摸或激活时的状态，可以定义普通状态下的背景色、文字颜色、边框等样式。
2、pressed（按下状态）：当用户按下控件时的状态，可以定义按下时的背景色、文字颜色、边框等样式。
3、focused（获取焦点状态）：当控件获得焦点时的状态，可以定义获取焦点时的背景色、文字颜色、边框等样式。
4、disabled（禁用状态）：当控件被禁用时的状态，可以定义禁用时的背景色、文字颜色、边框等样式。

二、其他的状态
1、selected（选中状态）：当控件被选中时的状态，可以定义选中时的背景色、文字颜色、边框等样式。
2、checked（勾选状态）：当控件被勾选时的状态，常用于复选框、单选按钮等控件，可以定义勾选时的背景色、文字颜色、边框等样式。
3、activated（激活状态）：当控件被激活时的状态，适用于特殊场景下的状态变化。
```





### 第02节 案例代码

代码

```xml
<?xml version="1.0" encoding="utf-8"?>
<selector xmlns:android="http://schemas.android.com/apk/res/android">
    <item android:state_pressed="true">
        <bitmap android:src="@drawable/main_item_official_yes" />
    </item>

    <item android:state_pressed="false">
        <bitmap android:src="@drawable/main_item_official_no" />
    </item>
</selector>
```

## 第三章 图层 layer

### 第01节 基础使用

```
使用 <layer-list> 可以方便地创建多个图层并组合在一起，实现各种复杂的绘制效果，如渐变、阴影等。
可用于创建图层列表的 XML 标签。它允许将多个不同的图层组合在一起，以创建复杂的绘制效果

<layer-list>
	<item> ...第1个,显示在最底层... </item>
	<item> ...第2个,显示在中间层... </item>
	<item> ...第3个,显示在最顶层... </item>
</layer-list>

item 在上面的，会显示在图层的最底层 
item 在下面的，会显示在图层的最顶层 
```





### 第02节 案例代码

```xml
<?xml version="1.0" encoding="utf-8"?>
<layer-list xmlns:android="http://schemas.android.com/apk/res/android">
    <!--  底层图层, 使用 Shape 定义一个圆角矩形作为背景  -->
    <item>
        <shape android:shape="rectangle">
            <corners android:radius="@dimen/dp_0010" />
            <solid android:color="#FF0000" />
        </shape>
    </item>

    <!--  上层图层, 使用 drawable 资源作为按钮的图标  -->
    <item android:drawable="@drawable/main_item_official_no" android:gravity="center" />
</layer-list>
```

### 第02节 案例代码

布局文件

```xml
<?xml version="1.0" encoding="utf-8"?>
<FrameLayout xmlns:android="http://schemas.android.com/apk/res/android"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:background="@color/white">

    <androidx.appcompat.widget.AppCompatTextView
        android:layout_width="100dp"
        android:layout_height="100dp"
        android:layout_gravity="center"
        android:clickable="true"
        android:background="@drawable/bg_button_demo"
        android:padding="@dimen/dp_0010" />

</FrameLayout>
```



综合案例

```xml
<?xml version="1.0" encoding="utf-8"?>
<selector xmlns:android="http://schemas.android.com/apk/res/android">

    <item android:state_pressed="true">
        <layer-list>
            <!--  底层图层, 使用 Shape 定义一个圆角矩形作为背景  -->
            <item>
                <shape android:shape="rectangle">
                    <corners android:radius="@dimen/dp_0010" />
                    <solid android:color="#00FF00" />
                </shape>
            </item>

            <!--  上层图层, 使用 drawable 资源作为按钮的图标  -->
            <item android:drawable="@drawable/main_item_official_no" android:gravity="center" />
        </layer-list>
    </item>
    
    <item android:state_pressed="false">
        <layer-list>
            <!--  底层图层, 使用 Shape 定义一个圆角矩形作为背景  -->
            <item>
                <shape android:shape="rectangle">
                    <corners android:radius="@dimen/dp_0010" />
                    <solid android:color="#FF0000" />
                </shape>
            </item>

            <!--  上层图层, 使用 drawable 资源作为按钮的图标  -->
            <item android:drawable="@drawable/main_item_official_no" android:gravity="center" />
        </layer-list>
    </item>
</selector>
```
