# IDEA 取消双击Shift键





[[TOC]]







## 第一章 旧版本设置

### 第01节 方法一

```
ctrl + shift + alt + /
	--------->  registry
	---------------------------->  勾选  "ide.suppress.double.click.handler"
```





### 第02节  方法二

```
ctrl + shift + t
	--------->  actions
	------------------------>  registry
	-----------------------------------------> 勾选  "ide.suppress.double.click.handler"
```











## 第二章 新版本设置

### 第01节 设置方式

```
1、 Settings
2、 Advanced Settings
3、 Disable double modifier key shortcuts

这里需要勾选 Disable double modifier key shortcuts 才能取消双击 Shift 按键的效果
```
