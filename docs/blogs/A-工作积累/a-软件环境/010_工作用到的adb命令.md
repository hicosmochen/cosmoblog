## 第一章 基础命令

### 第01节 常规命令

简单命令

```
adb  root
adb  remount

adb  devices
adb  shell
```



安装卸载包命令

```
adb  install  -r  -d   xxx.apk

adb  uninstall  com.xxx.xxx
```



推拉命令

```
# 将文件从 Android设备 拷贝到 电脑
adb  pull   Android设备的路径     电脑的路径
adb  pull  /data/system/dropbox   C:\crash\dropbox


# 将文件从 电脑 拷贝到 Android设备
adb  push   电脑路径    Android设备路径
adb  push   C:\crash\dropbox\Demo.apk     /system/app/demo/
```







### 第02节 私有路径

```
我的APK名称是  HonzonSystemUI.apk
我的APK包名是  com.hozonauto.systemui
推包操作 + 查看包信息 + 杀死进程


adb  root
adb  remount
adb  push  HonzonSystemUI.apk  /system/priv-app/HonzonSystemUI/
adb  shell  stat   /system/priv-app/HonzonSystemUI/HonzonSystemUI.apk
adb  shell  pidof  com.hozonauto.systemui
adb  shell  kill   3568
```







### 第03节 切换日夜模式

```
adb   shell   cmd   uimode   night   no

adb   shell   cmd   uimode   night   yes
```













## 第二章 其他命令

### 第01节 无线连接

```
adb root
adb remount

adb shell ifconfig wlan0

adb tcpip 5555
adb connect 192.168.0.32:5555

adb disconnect
```
