## 第一章 目录文件

### 第01节 目录操作

| 命令  | 作用                       | 使用案例                                                     |
| ----- | -------------------------- | ------------------------------------------------------------ |
| `pwd` | 显示当前工作目录的绝对路径 | `pwd`                                                        |
| `ls`  | 列出目录的内容             | `ls -al` 显示所有文件（包括隐藏文件）以及详细信息            |
| `cd`  | 切换目录                   | `cd /` 切换到根目录 <br>`cd ~` 切换到用户目录<br>`cd ..` 返回到上一层目录<br>`cd aosp/android-10` 进入到 aosp下面的 android-10 目录 |









### 第02节 文件操作

| 命令         | 作用                                   | 示例                                                         |
| ------------ | -------------------------------------- | ------------------------------------------------------------ |
| `touch`      | 创建空文件或者更新文件时间戳           | `touch  file.txt`                                            |
| `cp`         | 复制文件或者复制目录                   | `cp  source.txt  dest.txt` 复制文件 <br>`cp  -r  dir1  dir2` 复制目录 |
| `mv`         | 移动/重命名文件夹或目录                | `mv  old.txt  new.txt` 重命名<br>`mv  file.txt  /temp/` 移动 |
| `rm`         | 删除文件或目录                         | `rm  file.txt`  删除文件<br>`rm  -rf  dir/` 强制递归删除（谨慎使用） |
| `cat`        | 查看文件的内容（全部输出）             | `cat  /etc/passwd`                                           |
| `less`       | 分页查看文件内容（支持上下翻页）       | `less  large.log`  （按 `q` 退出）                           |
| `head`       | 查看文件中开头若干行                   | `head  -n  20  data.csv`                                     |
| `tail`       | 查看文件末尾若干行，常用于实时监控日志 | `tail -f  /var/log/syslog`  （持续跟踪新内容）               |
| `nano / vim` | 文本编辑器                             | `nano  config.ini`  或者 `vim  script.sh`                    |











### 第03节 文件权限

文件的权限系统 和 所有权

| 命令    | 作用                         | 示例                                                   |
| ------- | ---------------------------- | ------------------------------------------------------ |
| `chmod` | 修改文件权限（读、写、执行） | `chmod  755  scrip.sh`  （所有者 rwx  组 rx  其他 rx） |
| `chown` | 修改文件所有者               | `chown  alice:staff  file.txt`   （用户:组）           |
| `umask` | 设置默认权限掩码             | `umask  022`  使新文件默认权限为 644                   |

权限表示：  `rwx  r-x  r--`  对应数字 `754`  （ r=4，w=2，x=1 ）















## 第二章  文本进程

### 第01节 文本处理

| 命令   | 作用                           | 示例                                                         |
| ------ | ------------------------------ | ------------------------------------------------------------ |
| `grep` | 在文件中搜索匹配模式           | `grep  "error"   log.txt`  在 log.txt 当中搜索 error<br>`grep  -r  "TODO"  src/`  递归搜索 |
| `find` | 按照名称、类型、时间等查找文件 | `find  /home  -name  "*.py"`  查找home目录下面名称后缀是 py 的文件<br>`find  .  -mtime  -7`   查找最近 7天 修改的文件 |
| `wc`   | 统计行数、单词数、字符数       | `wc  -l  file.txt`  只显示行数                               |
| `sort` | 对文本行进行排序               | `sort  -n  numbers.txt` 按数字排序                           |
| `uniq` | 去重复行数（通常先排序）       | `sort  data.txt | uniq  -c`  统计重复次数                    |
| `cut`  | 按列截取文本                   | `cut  -d:  -f1  /etc/passwd`  截取用户名                     |
| `awk`  | 强大的文本分析工具             | `awk  '{print $1,$3}'  data.txt`   打印 第1列 和 第3列       |
| `sed`  | 流编辑器，用于替换、删除等     | `sed  's/old/new/g'  file.txt`  全局替换                     |







### 第02节 进程管理

| 命令    | 作用                                | 示例                                          |
| ------- | ----------------------------------- | --------------------------------------------- |
| `ps`    | 显示当前进程快照                    | `ps  aux` 显示所有进程详细信息                |
| `top`   | 实时显示进程资源占用（按 `q` 退出） | `top  -u  alice`  只查看用户 alice 的进程     |
| `htop`  | 增强版 top （需要安装）             | `htop`                                        |
| `kill`  | 终止进程                            | `kill  -9  1234`  强制杀死 PID 为 1234 的进程 |
| `nohup` | 使进程在退出终端后，继续运行        | `nohup  python  script.py &`                  |
| `&`     | 后台运行命令                        | `sleep  100  &`                               |









### 第03节 常见案例

查找占用CPU最高的进程

```shell
ps aux --sort=-%cpu | head -5
```















## 第三章 系统命令

### 第01节 磁盘存储

| 命令    | 作用                        | 示例                                         |
| ------- | --------------------------- | -------------------------------------------- |
| `df`    | 查找磁盘分区使用情况        | `df  -h`    以人类可读格式显示               |
| `du`    | 统计文件、目录 占用磁盘大小 | `du  -sh  /home/*`  显示每个用户目录的总大小 |
| `mount` | 挂载文件系统                | `mount   /dev/sdb1   /mnt/usb`               |
| `fdisk` | 磁盘分区工具                | `sudo  fdisk   -l`   列出所有磁盘分区        |











### 第02节 网络相关

| 命令          | 作用                     | 示例                                |
| ------------- | ------------------------ | ----------------------------------- |
| `ping`        | 测试网络连通性           | `ping  -c  4  google.com`           |
| `ifconfig/ip` | 查看、配置 网络接口      | `ip  addr  show` 显示IP地址         |
| `netstat/ss`  | 查看网络连接、接口监听   | `ss  -tuln`  显示所有 监听端口      |
| `curl`        | 发送 HTTP 请求           | `curl  -l  https:`                  |
| `wget`        | 下载文件                 | `wget  https:`                      |
| `ssh`         | 远程登录                 | `ssh  user@192.168.1.100`           |
| `scp`         | 安全复制文件（基于 SSH） | `scp  file.txt  user@remote:/temp/` |











### 第03节 系统信息

| 命令     | 作用                         | 示例                           |
| -------- | ---------------------------- | ------------------------------ |
| `uname`  | 显示系统内核信息             | `uname  -a`  显示全部信息      |
| `whoami` | 显示当前用户名               | `whoami`                       |
| `id`     | 显示 用户ID 和 组ID          | `id`                           |
| `date`   | 显示、设置 系统时间          | `date + %Y-%m-%d %H:%M:%S`     |
| `man`    | 查看命令手册（最强大的帮助） | `man  ls`  查看`ls` 的详细用法 |
| `--help` | 命令自带简短帮助             | `ls  --help`                   |
