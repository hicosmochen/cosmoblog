## 第一章  函数

### 第01节 基础说明

什么是函数？

```
1、C 程序是由函数组成的，我们写的代码都是由主函数 main()开始执行的。
2、函数是 C 程序的基本模块，是用于完成特定任务的程序代码单元。
```

函数的分类

```
1、系统函数，也叫做 库函数：这是由编译系统提供的，用户不必自己定义这些函数，可以直接使用它们，如我们常用的打印函数printf()。
2、用户定义函数：用以解决用户的专门需要。
```







### 第02节 库函数使用

**库函数使用步骤**

```
1、导入头文件
2、使用库函数  例如: printf()   scanf()
```

库函数的入门案例

```c
/**
    第一步 需要先在控制台中执行下面的代码, 进行编译操作
    gcc studyCDemo.c -o studyCDemo
    第二步 需要在控制台中执行下面的代码, 进行运行的操作
    studyCDemo
 */
#include <stdio.h>
#include <time.h>
#include <stdlib.h>
int main(){

	// 需要导入头文件 time.h
	// 功能: 得到系统时间  参数：常设置为NULL  返回值：当前系统时间
	// time_t 相当于long类型，单位为毫秒
	time_t tm = time(NULL);    
	
	// 需要导入头文件 stdlib.h
	// 功能：用来设置rand()产生随机数时的随机种子  参数: 如果每次seed相等，rand()产生随机数相等  返回值: 无
	// 随机种子, 只需要设置一次即可。
	srand((unsigned int) tm);  
	
	// 需要导入头文件 stdlib.h
	// 功能：返回一个随机数值  参数：无   返回值：随机数
	int r = rand();
	
	printf("r = %d \n", r);

    return 0;
}
```

### 第03节 函数定义格式

函数定义的一般形式：

```
返回类型 函数名(形式参数列表) {
	数据定义部分;
	执行语句部分;
}
```

**函数名**

```
理论上是可以随意起名字，最好起的名字见名知意，应该让用户看到这个函数名字就知道这个函数的功能。
注意，函数名的后面有个圆换号()，代表这个为函数，不是普通的变量名。
```

**形参列表**

```
在定义函数时指定的形参，在未出现函数调用时，它们并不占内存中的存储单元。
因此称它们是形式参数或虚拟参数，简称形参，表示它们并不是实际存在的数据，所以，形参里的变量不能赋值。
```

**函数体**

```
花括号{ }里的内容即为函数体的内容，这里为函数功能实现的过程，这和以前的写代码没太大区别，以前我们把代码写在main()函数里，现在只是把这些写到别的函数里。
```

**返回值**

```
说明:
	函数的返回值是通过函数中的return语句获得的，return后面的值也可以是一个表达式

注意:
	1、尽量保证return语句中表达式的值和函数返回类型是同一类型。
	2、如果函数返回的类型和return语句中表达式的值不一致，则以函数返回类型为准，即函数返回类型决定返回值的类型。对数值型数据，可以自动进行类型转换。
	3、return语句的另一个作用为中断return所在的执行函数，类似于break中断循环、switch语句一样。
	4、如果函数带返回值，return后面必须跟着一个值，如果函数没有返回值，函数名字的前面必须写一个void关键字，
		这时候，我们写代码时也可以通过return中断函数(也可以不用)，只是这时，return后面不带内容( 分号“；”除外)。
```





### 第04节 四个概念

**形参 和 实参**

```
形参:
	A. 位置: 写在函数定义的小括号里面的参数, 叫做形式参数, 也叫做 形参。
	B. 作用: 给出一套函数使用的标准。
	
实参:
	A. 位置: 写在函数调用的小括号里面的参数, 叫做实际参数, 也叫做 实参。
	B. 作用: 实际参与运算的数据
```

**局部变量 和 全局变量**

局部变量 也叫做 auto自动变量(auto可写可不写)

```
局部变量:
	A. 位置: 写在 函数内部 或者 复合语句的内部
	B. 特点: 随着函数调用的结束或复合语句的结束局部变量的声明声明周期也结束
	C. 赋值: 如果没有赋初值，内容为随机
	
全局变量:
	A. 位置: 在函数外定义，可被本文件及其它文件中的函数所共用，若其它文件中的函数调用此变量,须用 extern 声明
	B. 特点: 全局变量的生命周期和程序运行周期一样
	C. 要求: 不同文件的全局变量不可重名
```

### 第06节 递归函数

题目

```
计算 5 的阶乘

5! = 5x4x3x2x1 = 120
```

分析

```
5! = 5 x (4!)		5的阶乘等于 5乘以4的阶乘
4! = 4 x (3!)		4的阶乘等于 4乘以3的阶乘
3! = 3 x (2!)		3的阶乘等于 3乘以2的阶乘
2! = 2 x (1!)		2的阶乘等于 2乘以1的阶乘
1! = 1			    1的阶乘等于 1
```

递归概念

```
1、递归概念: 函数自己调用自己, 就叫做  "递归"
2、递归作用: 将一个大问题, 拆分成为相同的小问题, 将小问题解决了, 大问题就可以跟着解决。
```

**案例代码**

```c
/**
    第一步 需要先在控制台中执行下面的代码, 进行编译操作
    gcc studyCDemo.c -o studyCDemo
    第二步 需要在控制台中执行下面的代码, 进行运行的操作
    studyCDemo
 */
#include <stdio.h>

// 阶乘函数的定义
int jieCheng(int n){
	if(n == 1){
		return 1;    			 // 出口
	}
	return n * jieCheng(n - 1);  // 规则
}

int main(){
 
	int num = 5;
	int result = jieCheng(num);
	
	printf("result = %d \n", result);
    return 0;
}

```

运行结果

> result = 120

### 第02节 宏常量

场景

```
如果程序中, 大量使用到了 100 这个值, 那么为了方便管理, 我们可以将其定义成为 宏。
```

格式

```c
#define num 100
```

注意

```
1、宏名一般用大写，以便于与变量区别；
2、宏定义可以是常数、表达式等；
3、宏定义不作语法检查，只有在编译被宏展开后的源程序才会报错；
4、宏定义不是C语言，不在行末加分号；
5、宏名有效范围为从定义到本源文件结束；
6、可以用#undef命令终止宏定义的作用域；
7、在宏定义中，可以引用已定义的宏名；
```

案例代码

```c
/**
    第一步 需要先在控制台中执行下面的代码, 进行编译操作
    gcc studyCDemo.c -o studyCDemo
    第二步 需要在控制台中执行下面的代码, 进行运行的操作
    studyCDemo
 */
#include <stdio.h>

#define PI 3.1415926

int main(){
	double r = 10.0;
	double s = PI * r * r;
	
	printf("s = %lf \n", s);
    return 0;
}
```

运行结果

> s = 314.159260







### 第03节 宏函数

场景

```
在项目中，经常把一些短小而又频繁使用的函数写成宏函数，这是由于宏函数没有普通函数参数压栈、跳转、返回等的开销，可以调高程序的效率。
```

格式

```c
#define SUM(x,y)(x + y)
```

注意

```
1、宏的名字中不能有空格，但是在替换的字符串中可以有空格。ANSI C允许在参数列表中使用空格；
2、用括号括住每一个参数，并括住宏的整体定义。
3、用大写字母表示宏的函数名。
4、如果打算宏代替函数来加快程序运行速度。假如在程序中只使用一次宏对程序的运行时间没有太大提高。
```

案例代码

```c
/**
    第一步 需要先在控制台中执行下面的代码, 进行编译操作
    gcc studyCDemo.c -o studyCDemo
    第二步 需要在控制台中执行下面的代码, 进行运行的操作
    studyCDemo
 */
#include <stdio.h>

#define SUM(x,y)(x + y)

int main(){
 
	int a = 10;
	int b = 20;
	int result = SUM(a, b);
	printf("result = %d \n", result);

    return 0;
}
```

运行结果

> result = 30









### 第04节 文件包含

文件包含处理

```
1、基本概念：
	"文件包含处理" 是指一个源文件可以将另外一个文件的全部内容包含进来。Ｃ语言提供了#include命令用来实现 "文件包含" 的操作。
	
2、举例说明:
	A. 存在文件 file1.c 这个文件。
	B. 存在文件 file1.h 这个文件。
	C. 如果在 file1.c 文件当中, 想要使用 file1.h 文件的内容。
	D. 那么需要在 file1.c 的头文件, 引入:   #include "file1.h"
```

关于 include 的两种写法的区别

```
两种写法:
	1、 #include <stdio.h>
	2、 #include "stdio.h"
	
区别:
	1、如果采用的是 <> 常用于 包含 库函数的头文件。
	2、如果采用的是 "" 常用于 包含 自定义的头文件。

原理:
	1、采用 <> 表示系统直接按照系统指定的目录检索。
	2、采用 "" 系统会先在 file1.h 当中索引, 如果找不到的情况下, 再按照指定的目录检索。
```
