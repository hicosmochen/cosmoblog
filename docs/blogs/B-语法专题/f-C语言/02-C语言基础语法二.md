### 第04节 break 语句

描述

```
1、当它出现在switch条件语句中时，作用是终止某个case并跳出switch结构。
2、当它出现在循环语句中，作用是跳出当前内循环语句，执行后面的代码。
3、当它出现在嵌套循环语句中，跳出最近的内循环语句，执行后面的代码。
```

案例代码

```c
/**
    第一步 需要先在控制台中执行下面的代码, 进行编译操作
    gcc studyCDemo.c -o studyCDemo
    第二步 需要在控制台中执行下面的代码, 进行运行的操作
    studyCDemo
 */
#include "stdio.h"
int main(){

	int i = 0;
	while(1){
		i++;
		printf("i = %d \n", i);
		
		if(i == 10){
			break;  // 跳出 while 循环外
		}
	}
	
	int flag = 0;
	int m = 0;
	int n = 0;
	for(m = 0; m < 10; m++){
		for(n = 0; n < 10; n++){
			if(n == 5){
				flag = 1;
				break;    // 跳出for(n=0;n<10;n++)
			}
		}
		
		if(flag == 1){
			break;       // 跳出 for(m=0; m<10; m++)
		}
	}
	
	
    return 0;
}
```

### 第05节 continue 语句

描述

```
在循环语句中，如果希望立即终止本次循环，并执行下一次循环，此时就需要使用continue语句。
```

案例代码

```c
/**
    第一步 需要先在控制台中执行下面的代码, 进行编译操作
    gcc studyCDemo.c -o studyCDemo
    第二步 需要在控制台中执行下面的代码, 进行运行的操作
    studyCDemo
 */
#include "stdio.h"
int main(){

	int sum = 0;   // 定义变量 sum
	
	for(int i=1; i<=100; i++){
		// 如果 i 是一个偶数, 执行if语句中的代码
		if(i % 2 == 0){
			continue;   // 结束本次循环
		}
		sum += i;     	// 实现 sum 和 i 的累加 
	}
	
	printf("sum = %d \n", sum);
	
    return 0;
}
```

运行结果

> sum = 2500

## 第三章 数组

### 第01节 基础说明

什么是数组?

```
在程序设计中，为了方便处理数据把具有相同类型的若干变量按有序形式组织起来——称为数组。
```

微观概念

```
1、数组就是在内存中连续的相同类型的变量空间。
2、同一个数组所有的成员都是相同的数据类型，同时所有的成员在内存中的地址是连续的。
```

数组分类

```
1、一个数组可以分解为多个数组元素：这些数组元素可以是基本数据类型或构造类型。
2、按数组元素类型的不同，数组可分为：数值数组、字符数组、指针数组、结构数组等类别。
3、通常情况下，数组元素下标的个数也称为维数，根据维数的不同，可将数组分为一维数组、二维数组、三维数组、四维数组等。
4、通常情况下，我们将二维及以上的数组称为多维数组。
```

举例说明

```c
int  a[10];
struct Stu boy[10];
char s[10];
char *p[10];
```





### 第02节 一维数组

定义和使用

```
1、数组名称符合标识符的书写规范 （数字、字母、下划线）
2、数组名称不能与其他变量名称相同，同一个作用域内是唯一的
3、方括号[] 中常量表达式, 表示数组元素的个数

int[3] 表示数组当中有 3个元素。
其中下标从 0开始计算， 因此3个元素分别是  a[0]、a[1]、a[2]
```

备注

	1、定义数组时[] 内最好是常量, 使用数组时[] 内 既可以是常量, 也可以是变量
	2、在定义数组的同时进行赋值，称为初始化。全局数组若不初始化，编译器将其初始化为零。局部数组若不初始化，内容为随机值。
	3、数组名是一个地址的常量，代表数组中首元素的地址。

### 第03节 字符串

描述

```
1、C语言中没有字符串这种数据类型，可以通过char的数组来替代
2、字符串一定是一个char的数组，但char的数组未必是字符串
3、数字0(和字符‘\0’等价)结尾的char数组就是一个字符串，但如果char数组没有以数字0结尾，那么就不是一个字符串，只是普通字符数组，所以字符串是一种特殊的char的数组。
```

案例代码 **字符数组结尾符**

```c
/**
    第一步 需要先在控制台中执行下面的代码, 进行编译操作
    gcc studyCDemo.c -o studyCDemo
    第二步 需要在控制台中执行下面的代码, 进行运行的操作
    studyCDemo
 */
#include "stdio.h"
int main(){

	// 普通的字符数组
	char c1[] = { 'c',' ','p','r','o','g'};
	// 输出
	printf("c1 = %s \n", c1);   // 乱码, 因为没有 '\0' 结束符
	
	// 以 '\0' 结尾的字符数组是字符串
	char c2[] = {'c',' ','p','r','o','g', '\0'};
	printf("c2 = %s \n", c2);  
	
	// 字符串处理以 '\0' 作为结束符, 后面的 'h','e','l','l','o' 不会输出
	char c3[] = {'c',' ','p','r','o','g', '\0', 'h','e','l','l','o', '\0'};
	printf("c3 = %s \n", c3);  
	
    return 0;
}
```

运行结果

> c1 = c prog??
>
> c2 = c prog
>
> c3 = c prog







案例代码**字符串初始化**

```c
/**
    第一步 需要先在控制台中执行下面的代码, 进行编译操作
    gcc studyCDemo.c -o studyCDemo
    第二步 需要在控制台中执行下面的代码, 进行运行的操作
    studyCDemo
 */
#include "stdio.h"
int main(){

	// 不指定长度, 没有 '\0' 结束符, 有多少个元素, 就有多长
	char buff1[] = {'a', 'b', 'c'};
	printf("buff1 = %s \n", buff1);   // 乱码
	
	// 指定长度, 后面没有赋值的元素, 自动补'\0'
	char buff2[100] = {'a', 'b', 'c'};
	printf("buff2 = %s \n", buff2);
	
	// 所有元素赋值0
	char buff3[100] = {0};
	printf("buff3 = %s \n", buff3);
	
	// char buff4[2] = {'1', '2', '3'};  // 数组索引越界
	char buff5[50] = {'1', 'a', 'b', '0', '7'};
	printf("buff5 = %s \n", buff5);
	
	char buff6[50] = {'1', 'a', 'b', 0, '7'};
	printf("buff6 = %s \n", buff6);
	
	// 使用字符串初始化, 编译器自动在后面补0, 常用
	char buff8[] = "agjdslgjlsdk";
	printf("buff8 = %s \n", buff8);
	
	// '\0' 后面最好不要连着数字, 有可能几个数字连接在一起, 刚好是一个专一字符。
	// 例如:  '\ddd' 八进制的转义字符    '\xdd' 十六进制的转义字符    '\012' 相当于 \n
	char str[] = "\012abc";
	printf("str == %s \n", str);

    return 0;
}
```
