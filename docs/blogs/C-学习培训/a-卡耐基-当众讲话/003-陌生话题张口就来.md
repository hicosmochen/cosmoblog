## 开篇引论

### 1、说话过程中，注意事项

```
【1】讲话简洁明了， 不需要过多的语气助词
【2】口头禅想着去替代，朗读训练培养语感
【3】情绪融入语句，让大家参数感情共鸣
```





### 2、常见的问题

```
【1】听到或者看到这个话题的时候，大脑出现一片空白
【2】陌生话题从来没有听说过
【3】听说过这个话题， 但是无法组织合适的语言
【4】自己准备的不够充分， 讲的不够精彩

备注： 我们看到物品， 可以引申其寓意
```

## 万能讲话公式

张口就来的万能讲话公式

```
【1】问候大家 （表明开始讲话，平复心情）
【2】自荐 （我是....来自于....）
【3】感谢 （很荣幸.......）
【4】观点 （我今天给大家分享......）
【5】陈述 （单一层次、多层次）
【6】总结 （所以.... 因此.....）
【7】结语 （谢谢大家，鞠躬）
```









## 思维模式搭建

```
【1】放松 （没有可能所有的话题，自己都可以讲述）
【2】理解 （不熟悉或者陌生的话题，字面意思理解含义）
【3】印象 （将出现在大脑里面第一个词第一个印象写出来）
【4】主题和观点 （根据话题第一印象明确话题和观点）
【5】描述 （围绕第一印象多层次展开话题，发散思维模式）
【6】总结 （主题 + 单程车描述 + 观点）
```











## 张口就来的升级模式

```
【升级一】 《层次分明》   逻辑脉络组织素材，多角度讲述话题，显得更加有力，更加厚重
【升级二】 《善用结合》   结合自己的经历， 名人名言， 名人经历或者别人的经历运用， 寓言故事， 智慧故事， 名人名言等
【升级三】 《传递情绪》   让话题产生情绪， "快乐"、"悲伤"、"喜悦" 等
```
