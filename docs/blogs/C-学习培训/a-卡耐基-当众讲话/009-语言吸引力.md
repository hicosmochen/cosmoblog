## 第一章 开篇引论

### 第01节 观念说明

```
语言表达的注意事项：在表达的上面，重要的不是观念的陈述，而是能够找到能够解释观念的生动描述
```



### 第02节 如何增强吸引力

```
1、语言具体 
		A. 事例、比喻、数据
		B. 合适的比喻对于专业领域的比喻，简单的方式讲述给别人，用具体的数据告诉给别人
		
2、感同身受
		讲话之前，尽可能的站在观众的角度去思考。
			A. 思考一： 他们想听什么？
			B. 思考二： 不要仅仅只是想着 我想讲什么？
		这样之后， 就会让我们讲话更具有吸引力， 我们把这样的讲话方式叫做 "感同身受" 或者叫做 "换位思考"
```



### 第03节 五觉技法

```
1、五觉含义: 
	A. 视觉
	B. 听觉
	C. 嗅觉
	D. 味觉
	E. 触觉
	
	
2、五觉说明: 
	A. 说明: 使用五觉的方法进行场景的刻画， 让人如临其境， 感同身受增强语言吸引力。
	B. 目的: 我所感受的就是观众所感受的
```







## 第二章 场景刻画

### 第01节 点面结合

场景清晰，点面结合

```
1、任何事情的发生都离不开特定的环境。
2、讲述着需要让听众明白 "当前场景的特点，同时了解事件，主角在当前环境中所处的位置"
3、这就是需要我们， 既要讲述大环境， 也要讲述小环境， 也就是 点面结合

一句话训练：
		人物、地点、做什么
```







### 第02节 动态描述

场景生动，动态描述

```
1、变化之前的画面
2、动态变化的过程
3、变化之后的画面
```





### 第03节 保留声音

保留声音，还原现场

```
1、整个事件的情节充满动感。
2、特别是 "人物对话" 应该尽量保留原样，避免采用转述的方式
3、这样有利于场景的还原
```





### 第04节 感觉描述

选择最重要的感觉描述

```
1、不需要 用所有的感受都去讲述
2、只需要 讲述最重要的感受
```
