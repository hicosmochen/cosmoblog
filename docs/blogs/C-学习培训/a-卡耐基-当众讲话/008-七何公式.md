## 开篇引论

### 1、讲话不吸引人的原因

```
【1】空话套话，言之无物
【2】述事杂乱无序， 听着费力
【3】观点不鲜明，或者没有观点

为什么别人讲话吸引力很强大 ？
		需要使用具体话语讲述
```



### 2、讲话有吸引力的秘诀

```
学会讲故事

【1】讲述自己的故事 （首选）
【2】讲述身边的故事 （报纸媒体新闻）
【3】讲述名人故事
【4】讲述哲理故事
```









## 七何公式

### 1、公式介绍

```
【1】何时 （什么时间）
【2】何地 （什么地点）
【3】何人 （什么人物）
【4】何事 （什么事情）
【5】何果 （什么结果）
【6】何因 （什么原因）
【7】何观 （什么观点）
```





### 2、使用技巧

```
【1】一句话讲述前面的五何
【2】何时何地开头简洁真实性强
【3】何果在前，何因在后。 卖关子设置悬念
【4】先有观点，然后寻找故事，最后再阐述观点
```





### 3、公式难点

```
【1】一句话讲述五何
【2】何果简洁，简述事件的结果或者简述事件的性质，以及事件的个人感受
【3】描述何因的时候， 细节的堆砌。 何因围绕事情的结果， 最后得出观点， 详细讲述细节。

讲述故事的过程中，记得 "换位思考"
试想想，如果我是听众， 我最想要知道的是什么 ？
观点不要太多， 导致太分散， 一个观点足以。
```





### 4、课后训练

```
每周积累一到两个故事， 并且按照七何公式去述说
```
