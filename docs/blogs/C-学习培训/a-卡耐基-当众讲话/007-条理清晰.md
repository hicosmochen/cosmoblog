## 开篇引论

### 1、复述问题

```
【1】观点核心不要出错
【2】自行处理文章， 不需要全部复述

讲话的目的： 正确传达信息给对方
```



### 2、什么是条理清晰

```
【1】说话有条理， 有逻辑
【2】语调合适， 说话流程， 发音准确， 没有多余的语气词， 口头禅 （每天朗读 15分钟发音清晰）
```



### 3、表达不清晰原因

```
【1】没有在说话前， 做好充分的准备。 （思考的准备，语音表达的准备）
【2】说话偏离主题， 说话无重点
【3】急于表达， 观点不清晰
【4】多余口头禅， 或者语气词
```

## 实用技巧

### 1、如何做到条例清晰

```
【1】主题明确
【2】逻辑清晰
【3】充分准备
【4】多加训练
```





### 2、条理清晰的方法

```
【1】时间顺序
【2】空间顺序
【3】逻辑顺序 （点线面体、春夏秋冬） 自然顺序
【4】层次关系 （并列关系、递进关系）

"重点突出" 讲述自己的观点时候， 去掉与观点不相关的部分。
```





### 3、层次关系

```
对于任何主题或者观点的阐述， 只要你善于发掘， 总有可能找到事物内在的层次关系。

关键词法： 观点只有一个, 层次可以有多个
```

## 条理公式

### 1、公式介绍

```
【1】概述:  对于所讲述的内容点题， 并且进行概述
【2】条理:  将所讲述的内容以条理形式进行表述
【3】归纳:  归纳总结属于自己的观点，加以述说
```



### 2、注意问题

```
【1】概述要简洁
【2】数字很重要
【3】用单层次结构
【4】标题有讲究
【5】总结应突出
```



### 3、标题设置

```
【1】在标题的字数应该尽量相同， 采用类似结构
【2】在标题上加上数字， 或者常用的逻辑词
【3】让自己的标题押韵， 或者使用成语为标题
【4】标题字数最好少而精
【5】每个标题的首字符， 串联起来表达一定意思
```
