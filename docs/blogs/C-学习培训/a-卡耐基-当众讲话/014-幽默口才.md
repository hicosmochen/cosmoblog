## 第一章 开篇引论

### 第01节 注意问题

讲述幽默的时候，需要注意的问题

```
1、 良好的铺垫
2、 设置悬念 （衬和跌的技巧）
3、 场景再现 （幽默形象的建立）  声音（语气、语调）  表情（肢体表情）
4、 控制情绪 （不要提前笑场）
5、 内容健康
```





### 第02节 幽默作用

幽默技巧在讲话当中的作用，为什么需要使用幽默？ 使用幽默又什么好处？

```
1、 营造轻松的氛围
2、 引出话题的主题
3、 传达思想的见解
```









## 第二章 幽默结构

### 第01节 积累幽默

应用幽默的三部曲， 如何积累幽默

```
1、 这个幽默故事， 说明什么道理？
2、 我在什么时候， 可以用到这个幽默？
3、 讲述这个故事， 包括自己的感谢。
```



### 第02节 讲述幽默

```
1、 演讲的主题
2、 演讲的正文 （将幽默植入）
3、 演讲结束
```











## 第三章 课后练习

### 第01节 练习要求

阅读理解，某篇文章

```
1、 运用复述的方式, 在家中对着试衣镜练习， 将镜子里面的自己当成听众。
2、 注意语言的表达、眼神、站姿、以及肢体动作的应用
3、 同时用手机录音, 直至你对录音讲述的效果满意为止
```







### 第02节 练习步骤

```
1、 课后向着家人、朋友、同事运用本节课， 所学习的技巧，做即兴演讲训练。
2、 将在课程上的实战训练，回家之后，再次对着镜子练习 1 遍
3、 熟记今天的学习素材
```
