## 开篇引论

### 1、开场白的作用

```
【1】吸引听众注意力，激发听众的好奇心 （借物开场）
【2】概述你讲话的主要内容
【3】向听众阐明你讲话的必要性 （讲话内容有启发）
```



### 2、开场白的分类

```
【1】议论开场
【2】自嘲与幽默
【3】提问开场
【4】故事开场
【5】名人名言
【6】建立关系 （套近乎）
【7】赞美开场 （套近乎）
【8】悬疑开场 （反向开场）
【9】承上启下
```









## 实用技巧

### 1、悬疑开场

```
【1】用一组触目惊心的数据引出话题
【2】用别人意想不到的方式引出话题
```



### 2、议论开场

```
议论事物引发观点
抽象概念的方式才会采取议论开场白多角度的论证自己的议论

例如： 《青春》 《梦想》 《幸福》
```





### 3、提问开场

```
【1】设问开场 （自问自答） 提出问题， 指向性要强， 稍作停顿， 留给听众更多的思考时间， 但是这个时间不能太长
【2】疑问开场 （只问不答） 开场白里面不会给出答案，但是在正文当中给出答案 
```





### 4、故事开场

```
只要是自己的亲身经历最能吸引人: 
		亲身经历 --> 他人经历 --> 名人名言 --> 哲理故事

使用故事开场的注意事项：
		【1】可以是古今中外的好例子或哲理故事。 最好不要使用电视电影故事，一句话概述电视
		【2】故事的内容不能是大家耳熟能详的， 应该是绝大多数没有听过的故事
		【3】开场白故事简略， 重点突出， 发人深思
```





### 5、名人名言开场

```
"旧词新解" 不能够只是讲述原来的含义， 而改造创新， 讲述属于自己的新看法
```
