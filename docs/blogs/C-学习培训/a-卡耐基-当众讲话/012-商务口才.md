## 第一章  颁奖词

### 第01节 结构说明

```
1、 人物背景： 背景和成绩相关
2、 人物成绩： 业绩无法被替代
3、 颁奖嘉宾： 存在关联详细介绍
```



### 第02节 注意事项

```
1、 用具体的细节表现真实， 人物的业绩可以更加细微具体
2、 颁奖词宣读， 需要霸气十足
```









## 第二章 答谢词

### 第01节  个人答谢

```
1、 谢： 向评奖机构表示感谢
2、 归： 将功劳归功于协助你的人
3、 用： 说明奖励对我产生的作用感恩
4、 谢： 再次以真诚感谢
```





### 第02节 公司答谢

```
1、 谢： 向主办方表示感谢
2、 赞： 对活动表示赞许
3、 期： 提出期望，展望未来
4、 谢： 再次以真诚感谢
```







## 第三章 酒宴祝词

### 第01节 结构说明

```
1、 场： 称谓点题，欢迎，感谢
2、 意： 意义或事由，与在座人的关系
3、 举： 号召大家共同举杯
4、 贺： 表示祝愿， 提示干杯
```



### 第02节 简单说明

```
1、 主人： 表达欢迎感谢
2、 主宾： 表达祝贺
```
