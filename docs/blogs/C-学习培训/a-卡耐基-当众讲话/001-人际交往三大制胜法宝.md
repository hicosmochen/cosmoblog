## 开篇引论

### 1、前篇

```
1、让赞美形成生活中的习惯
2、不必为失败寻找理由，要为成功寻找方法
3、思考故事给我们带来的观点和启发
```



### 2、人际交往的四个方面

```
【1】 语言表达
【2】 文字表达
【3】 肢体动作
【4】 情感表达

备注： 人际交往的目的 信息的准确传递
```





## 第一制胜法宝  自我介绍



自我介绍的实用技巧：

```
【1】慢报姓名
【2】拆字转换
【3】个人名片（家乡籍贯、性格特征、爱好特征、目前状况、工作学习） 让别人知道自己的特征
【4】再次强调（真诚、细节化、体现真诚）
```







## 第二制胜法宝 介绍他人

人物介绍实用技巧：

```
【1】人物信息整理归纳（鲜明特点，特别...）
【2】他与我的关系，他的姓名
【3】你对他的整体印象
【4】介绍他的特点
【5】总结
```









## 第三制胜法宝 公众发言

讲台实用技巧：

```
【1】步伐： 稳重沉着，拖泥带水，不要凌波微步
【2】站姿： 五点一线（头肩臀脚跟小腿肚） 抬头收下颚，收小腹，双脚与肩同宽（女性丁字步）
【3】眼神： 与听众交流（虚视、点视、扫视、环视）
【4】表情： 自然微笑
```







## 课后结语

沉浸在自己的讲话过程中，自己就会忘记紧张

```
讲话应该 "顶天立地"

【顶天】说话有正能量，不要一直传递负面消息
【立地】幽默风趣的讲述自己的观点
```
