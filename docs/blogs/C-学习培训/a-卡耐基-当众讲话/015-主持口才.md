## 第二章 核心内容

### 第01节 开场词

开场词的构成

```
1、 题： 介绍活动会议主题
2、 重： 介绍活动主题的重要意义 （背景和意义）
3、 人： 介绍嘉宾资历，介绍活动环节
4、 情： 由于主持活动都有现场感，主持风格都应该保持 和 活动主题一致 （活泼、诙谐、动情）
```

实例分析

```
法制节目 （电视访谈）
	1、 背景介绍
	2、 主题以及意义
	3、 嘉宾介绍
	
分析：
	1、当主持的内容关注的是背景，就采用背景介绍开始。
	2、当主持的内容关注的是嘉宾，就采用重点在嘉宾的位置上。
	3、使用发散的思维看待问题
```



### 第02节 衔接词

衔接词的功能

```
1、 设置悬念 （提问、疑问） 《卖关子》 采用间接的方式告诉他人
2、 承上启下 （上挂、下接） 听众感受他们之间的联系
```



### 第03节 双人主持

如何实现双人主持？

```
1、 两人设疑，从普通的生活引入话题。
2、 第一个问题非常重要， 就像抛砖引玉一样导入话题
3、 第一个问题只是引入话题
4、 注意： 一问一答的次数，不得超过 6次，3分钟
```

如何承上启下？

```
1、 三角形法则，千丝万缕。
2、 寻找两者之间的关联点， 发现前后词语的共同点。 不是直接 A->B 而是先 A->C C->B 
```





### 第04节 结束语

结束语的构成

```
1、 总结： 对于活动的过程收获，进行总结点评
2、 主题： 再次强调活动的主题
3、 感谢： 感谢到场的观众
```

结束语的要求

```
对于未来的过程，有深远影响，有启发意义 （思考）
```
