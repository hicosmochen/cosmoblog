## 第一章 辩论基础

### 第01节 原则介绍

```
1、 摆事实、讲道理 （直接简洁）
2、 事实： 统计学结论、心理学结论、科学报告、谚语名言
```



### 第02节 注意事项

```
1、 大家如果都熟知的事物， 不需要深入讲解
2、 从不同的层面寻找论据， 论述自己的观点的正确性  （例如：文学上、经济上、社会上、军事上、政治上）
```



### 第03节 辩论目的

```
1、目的： 辨明是非， 探求真理
2、忌讳： 我们观点需要站在 公正的角度，不可以 强词夺理
```







## 第二章 辩论技巧

### 第01节 技巧说明

立论铁， 思维快

```
1、 突出自己的优点
2、 掩盖自己的缺点
3、 当对方攻击自己的缺点时， 还要回击他
```





### 第02节 注意问题

懂得多种思维模式： 发散思维，逆向思维

```
1、 锻炼快速的思维
2、 了解辩论的方法和技巧
3、 加强语言的精炼和逻辑
```
