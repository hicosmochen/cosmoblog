# 话题无限 玩转交际







[[TOC]]







## 开篇引论

为什么我们找不到话题？

```
【1】缺乏主动性 （打破沉默）
【2】对话题的了解不深刻 （倾听 提问主动抛出问题给别人, 别人就会有成就感）
【3】缺乏必要的技巧
```









## 实用技巧

### 1、陌生人寻找话题实用技巧

```
【1】用心观察 （口音、着装、长相、气质等） 第一句话是真诚的赞美， 赞美还要说出理由
【2】耐心挖掘，话题转换，认真发掘，热点事件。 观察对方的表情和手势， 是否是合适的话题
【3】用心交谈，善于观察对方的感受，从而适当的调整谈话的策略，尊重对方，适当的话题，让对方感觉到舒服
【4】遇到自己不了解的话题，倾听和提问，但是不要提出菜鸟级别的问题，如果需要转换话题。

首先，肯定别人的讲话，给自己带来知识见识
其次，坦诚的表示出自己的不足之处
```





### 2、联想扩散法 无限话题法

```
【1】借物
			a. 场景： 选择某一件事物
			b. 重现： 看到事物之后， 展开联想 （联想光是自己讲是不行的，必须和对方互动）
			
【2】借言 （扩散法）
			对方说的一句话时候， 就是话题源
```







### 3、谈话当中的禁忌

```
【1】交谈的过程当中，不要涉及对方不愉快事情。 （敏感的事情，隐私的问题）
【2】与不熟悉的人交流时， 不询问对方的衣服质量、价格、首饰真假等问题
【3】社交场合，不说低俗无趣的话题
【4】不随便议论别人的宗教信仰， 不要妄加非议
```





### 4、与人交流注意事项

```
【1】真诚
【2】初次见面， 趣味性的原则
【3】深层次的交流有情感上的链接

备注： 有共同情感共鸣和波动， 是真正要学习达到的目标
```
