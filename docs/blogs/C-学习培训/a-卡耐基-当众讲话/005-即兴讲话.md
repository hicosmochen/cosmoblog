## 开篇引论

### 1、需要具备能力

```
需要具备敏捷的思维能力和敏锐的语言感应能力

【1】锻炼思维
【2】口语表达能力
```



### 2、即兴讲话特点

```
【1】篇幅短小精悍 （3分钟左右）
【2】时境感强 （对于特点问题发表看法）
【3】就事论事，有感而发
```

## 实战技巧

### 1、金线串珠

```
金线串珠 （大脑里面散乱的思维点） 

观点: 作为自己的观点，现有观点然后组织语言想法，把相关的词语串联起来。
例如: 黄河、白板、水瓶、黑熊
恰当思想， 表达出一个观点
```





### 2、准备技巧

```
【1】知识素养的储备
			a. 历史资料： 古今中外重要的历史事件、历史人物、特别是历史事件的背景、人物的故事要清楚
			b. 相似资料： 对当前国内外发生的重大事件， 重要人物有关的情况清楚， 多看新闻观察背后的资料， 名人博客形成观点
			c. 文学资料： 对于古典诗词中外经典文学作品
			d. 名人名言： 俗语、谚语、网络热点词 等
			
【2】素材运用的准备
			a. 形成观点： 我们对于一个事件对咬形成自己明确观点
			b. 适时讲述： 一定要讲述出来， 才知道自己讲过 
```







### 3、快速思维

```
【1】思考主题： 主题是讲话的灵魂
【2】思考观点： 观点正确能够被大家接受
【3】思考结构： 开场正文结尾是怎么样方式
【4】思考现场： 承上启下方式传递现场
```
