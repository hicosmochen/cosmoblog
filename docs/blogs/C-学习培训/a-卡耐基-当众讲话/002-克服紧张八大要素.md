# 克服紧张 八大要素





[[TOC]]





## 开篇引论



### 1、导致紧张的原因

```
【1】自我认知度 （对自己的认识是否正确）
【2】技能掌握度 （技能娴熟）
【3】听众认知度
```



### 2、超越自我，实行目标

```
【1】准备不充分 （心理准备 + 内容准备）
【2】害怕自己讲不好
【3】害怕听众，反应不佳
【4】之前有过失败的经历
【5】自我形象威胁论

注意力集中之处， 就是能量汇集之处。
```







## 六步思考



### 一思  我是谁？

```
思考问题 ：  
		我是谁？ 我要给听众一个什么样子的印象？
		讲话的时候， 为自己树立一个高大的形象， 为自己贴上正面的标签。

例如：
		我为自己贴上的标签是 《稳重》《幽默》《内涵》
		我们平时需要尝试以这种标签的形象， 在我们的生活当中。
		学会忘记自己紧张的情绪， 为自己贴上印象的标签。
```





### 二思 我今天讲话的内容是什么？

```
思考问题 ：
		我今天讲话的内容是什么？ 我需要得到什么结果 ？
		讲话的时候，需要告诉听众一个明确的目的，说话有深入的主题思想， 思想有深度含义。
```





### 三思 为什么听众需要听我讲话？

```
思考问题 ：
		为什么听众需要听我讲话 ？
		给听众一个 与之相关的理由。 （取悦听众，开发听众热情）
```





### 四思 听众可能会有哪些反对的意见？

```
思考问题 ：
		听众可能会有哪些反对的意见？
		设想每一种可能， 针对于每一种可能， 都去设计解决方案。
		听众思维是 《纠错信息》 听众最善于发现我们话语当中的错误信息， 反而对于正确的信息，不会做出详细的思索。
```





### 五思 我们如何讲述才能达到最佳效果？

```
思考问题 ：
		我们如何讲述， 才能达到最佳的效果 ？
		设置【惊喜】
		设置【悬念】
		设置【快乐】
		设置【痛苦】
```





### 六思 我们如何吸引听众下次再来听我讲话

```
思考问题 ：
		我们如何吸引听众，下次再来听我讲话 ？
		讲话有深度， 具有启迪意义， 说话目的性明确
```
