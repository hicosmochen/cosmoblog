## 第二章 技巧公式

### 第01节 金字塔公式

结构说明

```
1、建议方案  开门见山，直接提出方案或者摆明观点
2、理由步骤  分析原因，并且阐述方案，实施具体步骤，对方是怎么想的，打消对方顾虑。 "换位思考，排除疑虑"
3、效果分析  如果我们采取了行动 ....... 会出现 ....... 效果
4、推动行动  
```

结构总结

```
金字塔公式
	1、 提出建议
	2、 说明理由
	3、 效果分析必要性
	
经常思考对方是怎么想的， 提升自己的情商， 讲述别人愿意看到的情况
```







### 第02节 魔法公式

结构说明

```
1. 列举事例 （摆事实） 传达观点
			A. 在列举事例的时候， 越真实的情感，越能够打动听众，并且影响到听众
			B. 当你的话语不是只仅仅关注自身，而是关注其他人的时候，你自然能够顺利的说服对方，采纳你的建议。
			
2. 说出结论 （说结论） 具体号召
			A. 号召需要具体
			B. 号召需要容易做到

3. 强调理由 （必要性） 推动行动
			A. 进一步激发听众行动的欲望， 给他们足够的动力和理由。
			B. 推动他们去执行你的建议
```

结构总结

```
魔法公式:
		1、 摆事实
		2、 说结论
		3、 必要性
		4、 推动性
```

注意事项

```
1、 观点清晰
2、 事例典型
3、 理由充分
4、 促进行动
5、 不同的主题，侧重点不一样
```







### 第03节 使用范围

```
1、 当我们找得到合适的事例时， 我们选择 "魔法公式"
2、 当我们找不到合适的事例时， 或者说事例不够鲜明的情况下， 我们需要使用 "金字塔公式"
```
