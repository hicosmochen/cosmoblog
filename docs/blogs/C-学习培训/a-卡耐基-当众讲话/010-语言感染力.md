## 第一章 开篇引论

### 第01节现象介绍

```
如何增强语言的感染力?
		当我们讲话滔滔不绝的时候，我们的听众一个个都趴下了，他们都在睡觉，都在玩手机，都在交头接耳聊天。
```



### 第02节 讲话境界

```
1. 把情绪很好的传递给别人
2. 感染力是一种可以能够引起别人，产生相同思想力量，启发智慧，激励感情的能力，他是讲话的高境界
3. 我们讲话的最高境界是 "以情动人"
```



### 第03节 七种情感

```
人的情绪常见的七种情感：

1、喜：高兴的情绪 	"眉飞色舞"
2、怒：愤怒的情绪 	"暴跳如雷"
3、忧：忧伤的情绪 	"唉声叹气"
4、思：思念的情绪 	"朝思暮想"
5、惊：惊讶的情绪 	"心惊肉跳"
6、恐：恐惧的情绪 	"大惊失色"
7、悲：悲伤的情绪 	"满面愁容"
```







## 第二章 情感再现

### 第01节 内容动情

```
1、首先 感染到自己
2、其次 感染到他人
3、再次 令人回味
```



### 第02节 声音动情

```
1、声音的讲话不要都在一个频道上面
2、声音传递包含着感情的基调
3、用我们的情感来还原当时的场景
```



### 第03节 表情动作

```
1、情由心生
2、清晰自然
3、悲喜有度
4、切忌矫揉造作
```



### 第04节 动作配合

表达含义的三个区域：

```
1、肩部以上的区域： 表达美好的赞美
2、躯干部分的区域： 表达一般性的陈述
3、腰部以下的区域： 表达贬低的含义
```

> 拳： 表示坚定的语气信心十足
>
> 掌： 上压外劈











## 第三章 实战训练

### 第01节 题目

```
我所经历的哪件事情，现在回想起来，现在回想起来，还很害怕 （生气、高兴、悲伤、感动 等）

1、事情的开始 __________________________________
2、事情的过程 __________________________________ （富含节奏，一波三折）
3、事后的感受 __________________________________
```





### 第02节 提示

```
例如：
	1、开车的时候，遇到了小型的车祸
	2、遭遇到小偷
	3、玩危险游戏
	4、看恐怖片
	5、走夜路
	6、被狗追
	
技巧：
	在选择话题素材的时候， 问自己几个问题：
	1、这个素材不一般吗？
	2、用 "七何" 和 "五觉" 的技巧如何讲述？
	3、我想表达的感情如何演绎？ 如 声音、表情、动作 等
```
