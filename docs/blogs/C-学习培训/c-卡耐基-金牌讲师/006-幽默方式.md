# 幽默方式

## 第一章 前言

### 第01节 幽默意义

```
1、意义：
		一个幽默的金牌讲师，一定能够让你的学习更加高效。

2、思考：
		怎样才能让课程更加幽默？
		
3、回答：
		整个课程体系当中，植入幽默的元素， 每一个环节都有幽默点
		
4、什么是幽默？
		幽默就是一种反差效果， 幽默的点不需要多， 只是在调动氛围
```









## 第二章 核心

### 第01节 培训加料

幽默为培训加点料

```
1、 火眼金睛找幽默：  人、事、言、情
2、 炉火纯青用幽默：  表达自然大方，表情语言夸张

培训课堂的幽默是从 培训课堂培训过程当中，现场植入的
```

> 如果调动不了幽默点， 需要调动氛围，课堂上需要多个演员，多个幽默点

常见的幽默方式

```
1、 生活类幽默
2、 男性女性类的调侃
3、 柿子比较软， 好欺负的学员
4、 模拟他人的动作
5、 偷偷的在背后说别人 （大家都熟悉的人）
6、 其他的幽默方式
```





### 第02节 幽默表现

```
1、 语言夸张，表达自然大方。 自然的演绎： 实力派 + 偶像派
2、 能学善用
```





### 第03节 设计幽默

设计幽默点， 贯穿整个课程当中

```
1、 引导01  ---> 穿插幽默点
2、 目标01
3、 引导02  ---> 穿插幽默点
4、 目标02
5、 引导03  ---> 穿插幽默点
6、 目标03
7、 引导04  ---> 穿插幽默点
8、 终极目标
```
