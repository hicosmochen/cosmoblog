### 第02节 KISS原则

```
1、 KISS 解释
		KEEP  IT  SIMPLE  AND  STUPID
		保持简单和愚蠢
		
2、简单
		A. 不要在 PPT 里面有长段的文字信息
		B. 能够用一句话就采用一句话
		C. 字尽量少
		D. 图文并茂，图大文少
		
3、视觉
		视觉上，需要一目了然，简单明了
```





### 第03节 强弱原则

```
1、口诀:
		遇强则弱，遇弱则强。
		学院的参与度强， 不要过度的展示 PPT 的效果
		
2、什么知识结构？
		能够让学员有所收获， 实实在在可以运用的 "干货" 着实落地，知识点不可以换， 非知识点可以换
```
