## 第一章 前言

### 第01节 概念

```
1、概念：
		谁拥有讲台， 谁就拥有未来
		
2、演讲和讲师：  演讲 < 讲师
		A. 演讲： 明事理，促行动，号召
		B. 讲师： 把我们会的传递给别人
		
3、天职：
		传达、授业、解惑
		传递思维， 帮他指引
		
4、讲师：
		把我们会的东西， 传递给别人， 让别人也能够学会
```



### 第02节 能力

``` 
1、说明：
		你认为你的能量可以达到什么的程度， 你就能达到什么样的程度
		
2、珍惜讲台
		A. 当讲师不同于演讲， 讲师一定要珍惜讲台
		B. 任何时候在听众没有产生膜拜之前， 都是挑剔的
		C. 成为不被人质疑的讲师， 成为能力的提升
		
3、成为讲师，需要具备什么能力？
		需要一个从 非讲师 变成 专业讲师 的能力
		
4、非专业讲师的性状
		A. 一说到培训，就准备 PPT
		B. 很多培训都是讲概念，做游戏玩氛围
		C. 认为嘴巴能讲，就能成为讲师
		D. 有些人当讲师，关注的是我能否表现出色
		E. 想当讲师，但是零基础，如何当讲师
```







## 第二章  要素定位

### 第01节 要素

金牌讲师的三要素

```
1、 专业技能： 我知道它了，我就是专业技能， 讲师需要把自己明白的知识点，变成知识体系
2、 授课技能： 把专业技能，转换成为别人会的能力
3、 演讲能力： 没有演讲的能力，没法当好一名讲师
```





### 第02节 定位

讲师的定位主要从三个方面

```
1、 个人背景  自己的经历和能力
2、 企业背景  企业的发展和历史
3、 社会背景  社会的需求以前和现在的方式
```
