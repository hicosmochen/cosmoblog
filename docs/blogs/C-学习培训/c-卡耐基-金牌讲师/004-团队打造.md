## 第一章 步骤

### 第01节 团队激励

> 分组， 分团队的目的是为了 激起竞争意识，竞争性、刺激性、被激动



团队打造

```
1、 破冰看热情度  （一动就有热情）
2、 看学习积极度  （学院的鼓励）
3、 游戏看活跃度
4、 运用看灵活度  （形变神不变）
5、 成果看应用度
```

两个荣誉

```
1、 集体荣誉感
2、 个人魅力度
```

> 活动是手段，授课是目的









## 第二章 分组

### 第01节 原则

```
1、 以乱为主： 尽量打破原有的机制，互相之间不认识
2、 均等原则： 人数尽量相等

如果人数较多，可以选出副队长
```





### 第02节 要求

```
1、 队长
2、 队名 （把企业的名称包含在里面）
3、 口号 （需要融入团队文化，公司文化）
4、 队徽 （绘制图标，挑战四张纸， 绘制队标）
5、 队歌
```
