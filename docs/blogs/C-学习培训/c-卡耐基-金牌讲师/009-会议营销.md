## 第一章 前言

### 第01节 会销目标

瞬间爆发你的成交量

```
1、 培养听众的意愿 （需要拥有公众演员，一帮的托）
2、 激发听众的欲望
3、 推进销售的进度
4、 促进销售的签单 （不是讲师去完成，而是其他成员去做）
```

### 第03节 听众特点

```
1、 意图明确
2、 有所期待

听众是所有期待行为的驱动
```

会议营销的特点

```
1、 目标明确  销售VS购买
2、 羊群效应  现场VS氛围
```





### 第04节 培训区别

常规培训特点

```
1、 满足听众的需求
2、 内心技能
```

会议营销特点

```
1、 满足听众的需求  "内心购买"
2、 满足企业需求    "销售"
```











## 第二章 核心

### 第01节 技巧

会议营销的常用技巧

```
1、 给好处，送赠礼，授技巧
2、 多体验
3、 建立关系，老乡，朋友，亲人
4、 说权威，知识点少讲
5、 谈感受
```

> 会议营销， 尽量少讲知识点 【在权威】 ---》 烘托









### 第02节 杀手锏

会议营销的杀手锏

```
1、 火眼金睛： 找到意向顾客，煽动性较强的顾客
2、 三寸之舌： 燃爆需求，刺激消费
3、 邀请体验： 价值支撑，获得认同
```

> 只有到了才说交钱成交的问题，如果没有到成交，都要邀请体验







### 第03节 活动区别

讲师需要做需求分析，根据听众发现需求，会议营销的后面，专业知识的后面，都会有一个产品。

```
会议营销和常规活动的区别： 主题性活动的准备
		1、会议营销具有引导性
		2、普通的活动没有强烈的引导性
```
