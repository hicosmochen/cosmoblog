


### 第02节 讲师推介

推介的方式

```
1、开发需求【痛点】简单有力，尽量排比
2、效果制胜【精准】
3、讲师背景【权威】
4、氛围烘托【高亢】幽默、热情
```

案例

```
各位同学,大家上午好！首先,让我们来一次问候,好吗?
请问大家想成为金牌讲师吗? 想,很想,非常想.请问大家能成为金牌讲师吗?能,当然,肯定能!

好的,非常感谢大家的配合.在这里呢,请允许我做一个小小的自我介绍,我叫金鑫,是今天各位同学们的助教老师.
我一直认为呢,如果我们老师和同学之间,用昵称进行称呼,会是一件令人愉快,且十分贴心的感受。
那么,请问一下朱磊同学,你的昵称是什么呢?是不是叫磊磊?好吧,我先说我的。
我是小鑫鑫,大家看我的体型,肯定觉得这反差也太大了,是的,以前呢,
大家都叫我大鑫,我觉得不够萌,所以将大改成了小.

OK，带给代价快乐,是我的职责。接下来,我要向各位隆重的推荐一位重量级的人物,
首先他是我心目中的男神。
第一次见到他时，我觉得他很儒雅,博古通今。
第二次见到他时,我觉得他很幽默。
第三次见到他时,我被他精湛的演讲和授课能力震撼了。

今天,他将为我们揭开金牌讲师的神秘面纱,让从没有登过讲台的你,可以登上属于你的巅峰舞台。
他是谁呢？同学们,请你们大声的告诉我,他是谁.他就是我们的男神老师,李四老师,
请大家用最最热烈的掌声,邀请李四李老师上场。
```
