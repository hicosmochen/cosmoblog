## 第一章 课程开发

### 第01节 简介

说明

```
1、 即使是曾经上过的课程，再重新上课的时候，也需要做课程开发
2、 课程开发是一个从无到有的过程
```

步骤

```
1、 开发授课需求
2、 开发授课对象
3、 开发授课目的
4、 开发授课课题
```





### 第02节 核心

开发授课需求

```
1、步骤
		A. 发现问题
		B. 分析问题
		C. 培训需求
		
2、说明
		培训需求 = 期待的需求 - 实际需求
		
3、举例
		A. 领导想要解决的 
		B. 员工想要得到的 
		C. 市场想要什么样的 
		D. 你想得到的
		E. 竞争对手的现状
		F. 企业能力的现状
```

开发授课对象

```
1、 内部对象
2、 外部对象
3、 内部对象 + 外部对象

到底对于谁培训， 对于有需要的人提供帮助
```

开发授课目的

```
1、 期待的结果是什么样子？
2、 通过这次上课， 给予大家的是什么， 达到什么样的目的？
			A. 抽象目的： （高大上 务虚）希望可以通过这次课程成为培训讲师
			B. 具体目的： （实际点 务实）在培训过程当中能够做什么样的程度
			
备注：
		在培训之前，写出抽象目的和具体目的
```

开发授课课题

```
1、 将课题开发的够炫酷， 有吸引力， 期待感
2、 举例： 三天打造一流口才
```









## 第二章 课程设计

### 第01节 核心

超级课程设计

```
引导01 ---> 【目标01】  ---> 引导02 ---> 【目标02】 ---> 引导03 ---> 【目标03】  ---> 引导04  ---> 【终极目标】
```



### 第02节 操作

这么做课程设计

```
1、 做课程的开场游戏
2、 课程开场白，引导讲师，引导主题
```

如果不匹配当中的主题，那么有两种方式

```
1、 增加润滑剂 （适当增加过渡元素）
2、 踢到外面去 （不要放进来）
```
