## 第一章 开篇引论

### 第01节 企业精神领袖

在企业里面，存在的精神领袖

品三国， 案例：

> 1、下邳 张辽  劝降 关羽 投靠曹操  【说服成功】

```
张辽不卑不亢，对关羽说道：“你若战死，将有三大罪过！如果你一意孤行战死沙场，有负当年桃园结义、誓同生死的盟誓。
你死了，你大哥和三弟怎么忍心活下去？刘备死了，怎么复兴大汉基业？此为第一罪；
刘备托付家眷给你，你若战死，两个嫂子成了俘虏咋办？后果你应该能想到。你有负大哥重托，对得起他和嫂子吗？此二罪也；
你武艺超群，威震天下，又熟读《春秋》，精通历史，不帮助大哥匡扶汉室，却逞一时之勇非死不可，这不是匹夫行为吗？这是第三个罪名”。
```

> 2、麦城 诸葛瑾 劝降 关羽 投靠孙权 【说服失败】

```
诸葛瑾说："我现在奉吴侯君命，特地前来劝告将军。自古道 识时务者为俊杰，将军所统领的汉上九郡，都已经归属于他人。
只有这座孤城，内无粮草，外物救兵，危在旦夕。将军要听我诸葛瑾一言归顺吴侯，既可以重镇荆襄，又可以保全家眷，万望君侯仔细考虑考虑"
```





### 第02节 如果需要说服某人

```
如果需要说服某个人， 需要找到弱点， 比他强， 找到关心的重视的价值取向， 发现他的不足之处。

案例： 《我是演说家》 乐嘉说服
```











## 第二章 超级说服力

### 第01节 核心内容

```
1、 要说服一个人， 要看他的价值取向：
		A. 对于逐利者， 可以加以利诱之
		B. 对于重名誉之人， 以名加之
		
2、解释：
		你必须要知道对方心里在乎的是什么， 看重的是什么才能起到良好的说服效果
```





### 第02节 案例分析

案例一

```
题目：
		如何引进高效的人才
		
分析：
		1、我们的企业能够为他们创造什么平台
		2、不要被曾经的履历所蒙蔽， 价值观是否相同
		3、是否在平台上面创造一番事业
```

案例二

```
题目：
		下属能力比自己强
		
分析：
		1、管理型人才  （专业性人才的关系）
		2、管理型人才  （情商较高）
		3、专业性人才  （智商较高）
		
说服：
		关注弱点，当一方面较强的时候， 另外一方面必然较弱
```



案例三

```
题目：
		如何成为有说服力领导
		
分析：
		1、领导讲话，下属不听从
		2、朋友讲话，下属彼此之间听从
```

案例四

```
题目：
		如何管理 90 后员工
		
分析：
		1、领导者 和 员工 是否在同一个平台
		2、需要做的是领导者和员工处于同一个对等的平台，而不是彼此对立的平台上面
```
