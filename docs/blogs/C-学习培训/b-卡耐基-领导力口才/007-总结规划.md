## 第二章 核心内容

### 第01节 要点

总结与规划要点  "一全六有"

```
1、 全面性 （视角全面）
2、 有高度
3、 有深度
4、 有看点
5、 有遗憾
6、 有期待
7、 有情感
```



### 第02节 技巧

```
1、 重点突出： 核心关键词，突出重点，全面性概述
2、 事实依据： 与听众相关，略讲（一句话）看新闻的标题就是一句话，高度的概括，这种高度概括的好处，采用排比句式（语气强烈） 多句话概述高度的概括，多个层次
3、 情感传递： 巧用亮点，重在煽情，情绪融入讲话过程中，打动听众
4、 现场互动： 内容互动，眼神互动，交流互动
```



### 第03节 注意

```
1、 总结切记说成流水账
2、 总结不能文过饰非， 太文学范
3、 总结必须有亮点， 不能老生常谈

条理不清晰：需要框架模式的归纳
```
