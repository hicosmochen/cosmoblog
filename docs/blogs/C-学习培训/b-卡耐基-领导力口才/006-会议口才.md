```
会议发言遇到的问题：
	1、会议发言的时候紧张
	2、会议发言觉得没话可讲
```

### 第05节 主持人

```
1、到会时候， 如果人员太多， 直接快速发言， 在进行人员介绍的时候，不需要给观众鼓掌时间，快速切换下一位领导，最后在鼓掌
2、随时准备讲话， 开会的过程中，上下人员发言完毕之后，注意分为的调度，以及承上启下的人物穿插
```









## 第三章 讲话主动权

### 第01节 四大秘籍

```
1、 迫切性： 事情有缓急，找出迫切需要解决的问题，目前最迫切的问题
2、 简化性： 雾里看花，将复杂的问题简单化，站身事物之外看待问题，以旁观者眼光看待，需要注意 "当局者迷，旁观者清"
3、 完美性： 提出完美的解决方案， 提出自己对于问题的建议
4、 根源性： 从根本上解决问题，企业目标 "将利益最大化"  降低成本，提高效益
```



### 第02节 注意事项

会议过程当中，需要注意的问题

```
1、 讲述问题简洁化 "化繁为简"
2、 声音注意起伏
3、 提高自己的思想境界，提升自己身份地位
4、 根源问题，完美解决，有明确的指向性，提出切实可行的解决方案，具有可行性
5、 当自己气势强大的时候，如果别人的气势并不是特别强大，需要使用气势强硬的压制对方，精神控制 "稳重，有领导范"
```
