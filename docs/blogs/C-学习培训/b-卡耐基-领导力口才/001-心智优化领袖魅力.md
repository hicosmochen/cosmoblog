## 第二章 领导素质

### 第01节 讲话善用三性

```
1、 理性： "逻辑推理" 用清晰的逻辑说服人
2、 感谢： "感情融入" 用真挚的情感打动人
3、 知性： "哲理性" 用富含哲理的智慧打动人
```



### 第02节 四种钱以外能力

```
四种能力
		1、 做人的能力： 做人的姿态，为人处世
		2、 树立正确价值观能力： 尺度衡量决策
		3、 拥有足够的毅力和耐力
		4、 正确判断未来的能力

未来决策
		1、未来变化的趋势，需要有极强的判断力和决策力。
		2、大势所趋，走出和别人不同的路
```





### 第03节 素质模型

```
1、 思想模型： 领导者总是会把自己的思想，自己的智慧，放大成为大家的思想
2、 潜力挖掘： 领导者善于发掘他人的潜力，并且让他们的梦想得以实现
3、 社会价值： 领导者是成就他人的人， 当你成就了他人，也就成就了自己
```
