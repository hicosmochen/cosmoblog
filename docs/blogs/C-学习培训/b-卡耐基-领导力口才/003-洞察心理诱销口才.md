## 第二章 判断类型

### 第01节 团队成员

如何判断团队里面成员的类型？

```
多让团队成员参与活动， 持续性的做同一件事情， 观察人员的类型。

1、竞争型
2、成就型
3、自我欣赏型
4、服务型
```





### 第02节 顾客类型

如何激发顾客的购买欲望

```
1、理性顾客： 科学分析，专业知识讲解，用专业知识去讲述产品的功效
2、感性顾客： 谈心的时候，好感和信任感，感情牌
```
