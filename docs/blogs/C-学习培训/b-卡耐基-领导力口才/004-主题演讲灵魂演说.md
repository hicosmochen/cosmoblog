## 第二章 思想含义

### 第01节 演讲思想

用新的模式，打破常规的模式，运用主题演讲，颠覆传统

```
1、 为自己的主题讲话， 设置主题背景
		例如： 我们为酒店设置主题背景， 新加坡风格酒店，欧式风格酒店，美式风格酒店
		
2、发自内心的感觉
		认为自己是什么， 讲话就是什么
		A. 认为自己是高尚的，讲话的内容就是高尚的
		B. 认为自己是坚强的，讲话的内容就是坚强的
		C. 认为自己是诚信的，讲话的内容就是诚信的
```

### 第03节 主题含义

为讲话的过程中，设置主题思想，主题灵魂，讲话的灵魂

```
1、 政治实例
		为什么大家都去推崇习大大?
		"中国梦"  主题灵魂
		
2、回答
		A. 习大大为我们营造了一个 "中国梦" 实现我们中华民族的伟大复兴。
		B. 虽然说 "中国梦" 知识概念化的目标， 但是能够激发每一个人的奋斗热情。
		C. 因为 "中国梦" 的存在，让我们每一位中国国民，都燃起新的希望，新的人生目标
		
3、以往
		A. 相比以往的领导人  "三个代表"、 "八荣八耻"
		B. 只是看待问题的表面，以眼前现象为基础，表示当前的基本现状，却没有给出深入的核心，没有让身边的国民奋斗深入灵魂的深处，并没有激发每一个人深层次的意识。
		C. 而毛主席习大大等，却是能够激发广大劳苦人民站起来，当家做主，这是一个梦想，深入人心，这才是办实事，因此很多人都会为梦想奋斗。
```
