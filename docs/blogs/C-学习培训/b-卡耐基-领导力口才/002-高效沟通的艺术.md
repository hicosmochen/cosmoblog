```

		所谓的高效沟通，就是指： 我们能够找到问题的本质， 根据问题的本质去进行沟通， 这才是高效沟通
		
2、激发内心需求
```

### 第02节 个人魅力

个人魅力

```
1、 公式：
			个人魅力  =  外形  +  内涵  + 智慧
			
2、 解释：
			A. 外形： 个人的衣着服饰，梳妆打扮
			B. 内涵： 一个人的精神文化修养 （积累）
			C. 智慧： 一个人的知识面广泛程度
```

沟通细节

```
1、 基础细节
		A. 微笑： 从内心深处真正的微笑
		B. 眼神： 看向听众的脑门心 （眉心）
		
2、面对面沟通的有效距离
		A. 亲密关系： 亲人爱人  0.0m --- 0.5m 
		B. 舒适距离： 最佳沟通  0.5m --- 1.5m
		C. 疏远距离： 关系疏远  1.5m --- 3.0m
```

### 第02节 管理学案例

案例1

```
1、场景：
		A. 张三和李四拥有相同的才能，相同的工作性质，相同的人物背景。
		B. 现在身为领导者，必须提拔一名成为核心骨干
		C. 那么此时此刻，领导应该怎么说，才能让大家都心服口服
		
2、解决：
		A. 声明领导在某些方面， 必须提拔一名， 根据综合情况考虑。
		B. 比如人际交往口才，必须提拔一名
		C. 同时给出其他人提拔的机会
		
3、分析：
		A. 鱼与熊掌不可兼得，有时候舍弃其中，但是必须给舍弃一方希望。
		B. 鼓励员工只要能够不断地去完善自己，一样有提升的机会。
```



案例2

```
1、场景：
		A. 发现我的顶头上司出现严重工作失误的情况下，
		B. 采取越级上报的情况
		C. 应该如何向高级汇报情况
		
2、解决
		A. 谈论我本人越级上报的原因，顶头上司工作失职所在。
		B. 给高级领导一个合理的解决方案，而不是把问题抛给高级领导，让高级领导解决问题，需要给出相关合理的方案
		
3、分析
		A. 我们需要的是帮助上级解决眼前的问题
		B. 而不是只是指出上级的错误
		C. 强调需要 "解决问题" 而不是 "汇报问题"
```



案例3

```
1、场景：
		A. 小孙刚来到公司，因为个人优秀的原因，被提升为领导
		B. 但是手下的老员工，并不是很服从小孙的安排
		C. 这种情况下，小孙应该如何解决？
		
2、解决
		A. 寻找到老员工当中，领军人物。
		B. 与领军人物进行沟通交流
		C. 让领军人物带领其他人，改变眼前的状况
		
3、分析
		A. 擒贼先擒王，解决领头羊
		B. 让领头羊解决其他人
		C. 借力反施
```
