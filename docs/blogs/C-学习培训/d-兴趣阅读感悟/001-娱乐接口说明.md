## 第一章 小说接口

### 第一节 小说搜索

#### 第01部分 接口API

请求

```
GET
```

地址

```
https://api.pingcc.cn/fiction/search/{option}/{key}/{from}/{size}
```

参数

| 参数名称 | 参数类型 | 参数描述                                                     | 是否必填 |
| -------- | -------- | ------------------------------------------------------------ | -------- |
| `option` | `string` | 选择搜索项 <br />标题：title<br />作者：author<br />分类：fictionType | 是       |
| `key`    | `string` | 搜索关键字                                                   | 是       |
| `from`   | `int`    | 当前页数，留空默认 1                                         | 否       |
| `size`   | `int`    | 一页显示的数量，留空默认10，最多30                           | 否       |
